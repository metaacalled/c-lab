# c-lab

En este repositorio me dispongo a documentar mi proceso de aprendizaje en **C** desde "cero".

Mis objetivos son:

- Dominar la programación a bajo nivel.
- Dominar el uso de memoria.
- Estructuras de datos.
- Sobrevivir.

Mi objetivo es desarrollar una base sólida en C a través de **práctica diaria**, ejercicios progresivos y pequeños proyectos.

## Metodología
- Debo hacer todos los dias (o casi todos) un proyecto pequeño.
- En cada carpeta, voy a incluir:
  - Código limpio y comentado  
  - Explicación del ejercicio  
  - Notas personales
 
Si estás leyendo esto, seguramente no tengas nada mejor que hacer o tengas un mínimo interes en mi perfil profesional.
En ambos casos, hola.
